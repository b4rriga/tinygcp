# TinyGCP

## Description

A simple, extremely barebones G-code parser.
As is, this library can parse through a sequence of G-code _blocks_ (meaning commands in G-code parlance) and extract:
- M-codes
- G-codes
- X, Y and Z axis position
- Machine feed rate
- Comments (if any)
Adding support for more words, such as additional axes, should be trivial.
